# DoubleOreDrops
DoubleOreDrops is an increased ore drop and auto-smelting plugin.

Ores Increaseed;
-Iron
-Gold
-Coal
-Diamond
-Redstone
-Emerald
-Lapis

While increased the drop rate on each of these ores, Iron and Gold ore now drop ingots instead of Ore.
